# data1030-finalproject
This project focuses on building machine learning models to predict in-hospital mortality among critically ill patients using the SUPPORT2 clinical dataset. The goal is to assist clinicians and healthcare providers with data-driven insights that can support early decision-making and improve quality of care.

Predicting mortality risk in real time can help hospitals allocate resources more effectively, initiate timely interventions, and improve end-of-life care planning.

All analysis, modeling, and results presented in this repository are based on the methodology and findings summarized in the accompanying project presentation
